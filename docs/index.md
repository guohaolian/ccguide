## 为什么需要 V2？

2026年3月31日，Claude Code v2.1.88 的 **51.2 万行 TypeScript 源码** 意外泄露。这不是一个简单的 CLI 工具——这是一个以 LLM 为内核的操作系统。V2 基于这份完整源码，结合全网优质解读，打造最全面的学习指南。

### V2 相比 V1 的重大升级

| 对比项   | V1         | V2                                                              |
| -------- | ---------- | --------------------------------------------------------------- |
| 章节数   | 12篇 114节 | **20篇 200+节**                                                 |
| 文档框架 | Docsify    | **VitePress**（更快、暗色模式、全文搜索）                       |
| 新增篇章 | -          | 提示词工程、记忆系统、隐藏功能、Hooks生态、Token经济学、实战Lab |
| 图表数量 | ~30个      | **100+ Mermaid图 + 60+ 表格**                                   |
| 输出格式 | 网页       | **网页 + PDF + HTML离线包**                                     |
| 实战练习 | 少量       | **8个完整Lab**                                                  |
