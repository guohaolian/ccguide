# 下载

## PDF 版本

**[前往 GitHub Release 页面下载 PDF (带书签, ~28MB)](https://github.com/bcefghj/claude-code-complete-guide_v2/releases/tag/v2.0.0)**

PDF 版本包含全部 20 篇 187 节内容，带有完整书签导航。

::: tip 推荐在线阅读
在线版本包含交互式 Mermaid 架构图、全文搜索、暗色模式等功能，体验更佳。

[在线阅读](https://bcefghj.github.io/claude-code-complete-guide_v2/)
:::

### PDF 特点

- 完整书签导航（20 篇 + 附录）
- 页眉页脚含页码
- A4 排版，适合打印
- 代码高亮
- 精美封面

### 注意事项

- PDF 中的 Mermaid 图表显示为占位提示，请在在线版本中查看交互图表
- 表格和代码块已针对打印优化
- 建议使用 Adobe Acrobat 或系统自带 PDF 阅读器打开

## HTML 离线包

运行以下命令构建离线 HTML：

```bash
git clone https://github.com/bcefghj/claude-code-complete-guide_v2.git
cd claude-code-complete-guide_v2
npm install
npm run docs:build
# 打开 docs/.vitepress/dist/index.html 即可离线阅读
```
